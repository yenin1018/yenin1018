開始每日一文章

之前很少寫筆記，會寫也只是寫在空白筆記本上面，但會發現常常寫著就忘記寫在哪裡，要找以前的資料筆記就找不到，不可快速搜尋 Ctrl + F 尋找，翻東翻西就是找不到哈。

文章卡卡請見諒

大多筆記會先以自己看懂為優先，在從別的角度去延伸，很難一次就到位，難啊！只能先求有再求好，好可以再更好。

使用 MarkDown

第一次使用 MarkDown 寫文章，很多語法還不熟悉，雖然簡單，但還沒開始使用就會不知下一步該怎麼走。

加油吧！！
